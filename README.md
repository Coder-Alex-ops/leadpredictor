# LeadPredictor 🧮

Калкулатор за прогнозиране на нужните **контакти (prospects)**, **потенциални клиенти (leads)** и **клиенти (customers)** за постигане на зададен оборот в маркетингова кампания.

🔗 **Live demo:** _(добави твоя Netlify линк тук)_

## Формули

| # | Резултат | Формула |
|---|----------|---------|
| 01 | Клиенти (Customers) | `Общ оборот / Средна стойност на поръчка` |
| 02 | Потенциални клиенти (Leads) | `Клиенти × 100 / Процент отговори (leads)` |
| 03 | Контакти (Prospects) | `Потенциални клиенти × 100 / Процент отговори (prospects)` |

## Функционалности

- Жив изчислителен модел — обновява се при всяка промяна на вход или плъзгач
- Хоризонтална диаграма по месеци (Chart.js) с tooltip
- Превключване на език (English / Български) и валута ($ / € / лв.)
- Адаптивен (responsive) дизайн

## Технологии

Чист HTML, CSS и JavaScript + [Chart.js](https://www.chartjs.org/) (без build стъпка).

## Локално стартиране

Отвори `index.html` директно в браузъра, или:

```bash
python3 -m http.server 8000
# отвори http://localhost:8000
```
